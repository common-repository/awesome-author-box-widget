Awesome Author Box Wigets

 
== Description ==
 
Awesome Author Box Wigets display an author box to your theme sidebar/widget area or in Post/Page with Social media links, Signature. 5 common social links help user to connect with more people.

Easy to install
Fully integrated in WordPress
Compatible with WordPress 3.9 and PHP 5.6
Can be display both in Widget area or Post/Page
Attractive admin panel

Demo Link of this plugin: <a href="http://www.modinatheme.com/plugins/awesome-author-box-widget/">Live Demo</a>

== Installation ==
 Download and Install. Then Use it. Easy :D 
 
1. Upload `awesome-author-box-widget.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Widegts Area ('/plugins/wp-admin/widgets.php')
4. Find The "Awesome AUthor Box"
5. Use it and Enjoy.
